Installation

You should find the folder named
/Codecanyon-flutter-news-time
extract and copy it to your main folder of your domain directory
Now create a database and save the username, password and database name.
Now go to your phpmyadmin find your database name and import the sql file
Open
/Codecanyon-flutter-news-time/.env file
and edit this line:

APP_URL="localhost/"
with your url.

and change the databse name, username & password
DB_DATABASE=news_db
DB_USERNAME=root
DB_PASSWORD=

Firebase Server Key

Now Paste your firebase server key inside PushNotificationsController inside
app/Http/Controllers/PushNotificationsController.php
and change the firebase server key with your in line no. 37

SMTP Configuration

Go to your hosting server and locate Email Configuration and create an email account and save smtp host name, port, username, password, email and replace with below data

MAIL_MAILER=smtp
MAIL_HOST=smtp hostname
MAIL_PORT=2525
MAIL_USERNAME=username
MAIL_PASSWORD=password
MAIL_ENCRYPTION=null
MAIL_FROM_ADDRESS=fromEmail
MAIL_FROM_NAME="${APP_NAME}"
